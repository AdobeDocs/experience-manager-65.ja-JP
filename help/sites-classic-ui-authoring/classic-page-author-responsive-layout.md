# レスポンシブレイアウト{#responsive-layout}

>[!CAUTION]
>
>レイアウトコンテナコンポーネントは、クラシック UI でも使用できます。ただし、完全な機能がサポートされ、使用できるのは、タッチ対応 UI の場合のみです。詳しくは、標準オーサリングのドキュメントの[レスポンシブレイアウト](/help/sites-authoring/responsive-layout.md)を参照してください。
