# 複数のカテゴリページと製品ページの作成

[AEM CIF コアコンポーネント](https://github.com/adobe/aem-core-cif-components)を使用すると、コンテンツ作成者は、追加のカテゴリページや製品ページを作成できます。これらのページは、追加のマーケティングコンテンツで強化し、特定のカテゴリや製品をターゲットするように設定できます。次のビデオでは、この機能の設定について説明します。
