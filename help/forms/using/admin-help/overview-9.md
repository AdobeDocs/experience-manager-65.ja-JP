# Forms ワークフローの概要 {#overview-of-forms-workflow}

Formsワークフローを使用すると、ユーザーは人間中心のプロセスを設計、管理および最適化できます。 Forms ワークフローを使用して、システムとユーザーを接続する自動ビジネスプロセスアプリケーションを管理することもできます。
