# ユーザーまたはグループの検索 {#search-for-a-user-or-group}

1. 管理コンソールで、設定／User Management／ユーザーとグループをクリックします。
1. 検索条件を絞り込む情報を指定し、「検索」をクリックします。検索の結果がページの下部に表示されます。列見出しをクリックすると、リストを並べ替えることができます。最大 1000 件の結果が返されます。
