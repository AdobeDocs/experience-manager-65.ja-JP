# セキュリティ設定の指定{#configuring-security-settings}

パスワードを設定したり、印刷や編集など特定の機能を制限したりして、PDF ドキュメントへのアクセスを制限できます。PDF ドキュメントに制限された機能が含まれている場合、それらの機能に関連するツールやメニュー項目は淡色表示されます。また、ドキュメントの暗号化や認証など、他の方法を使用して保護されたドキュメントを作成することもできます。セキュリティ設定には、特定の PDF 変換に使用するパスワードと固有のオプションがあります。

セキュリティ設定ページでは、次のタスクを実行できます。

## セキュリティ設定の作成または編集  {#create-or-edit-a-security-setting}

*セキュリティ設定*&#x200B;によって、そのセキュリティ設定を使用して変換されたファイルのセキュリティと権限が制御されます。

1. 管理コンソールで、サービス／PDF Generator／セキュリティ設定をクリックします。
1. 「新規作成」をクリックするか、セキュリティ設定の名前をクリックします。
1. セキュリティ設定の新規作成／編集ページで、セキュリティ設定の必須情報を入力します（[ファイルタイプ設定の指定](/help/forms/using/admin-help/configuring-file-type-settings.md#configuring-file-type-settings)を参照）。
1. 「保存」をクリックし、表示されたダイアログボックスに設定の名前を入力して、「OK」をクリックします。

### セキュリティ設定 {#security-settings}

次の設定で、互換性と暗号化を設定します。セキュリティ設定へのアクセス方法については、[セキュリティ設定の作成または編集](configuring-security-settings.md#create-or-edit-a-security-setting)を参照してください。

**互換性：パスワードで保護されたドキュメントを開くための暗号化の種類を** 設定します。「Acrobat 3.0 以降」オプションでは、低い暗号化レベルを使用し、その他のオプションでは高い暗号化レベルを使用します。

**Acrobat3.0以降：低い暗号化（40ビットRC4）** を使用します。

**Acrobat 5.0以降：高い暗号化（128ビットRC4）** を使用します。

**Acrobat 6.0以降：高い暗号化（128ビットRC4）** を使用します。このオプションでは、メタデータで検索を有効にできます。

**Acrobat 7.0以降：高い暗号化（128ビットAES）** を使用します。このオプションでは、メタデータでの検索と、添付ファイルのみの暗号化を有効にできます。

**Acrobat 9.0以降：高い暗号化（256ビットAES）** を使用します。このオプションでは、メタデータでの検索と、添付ファイルのみの暗号化を有効にできます。

古いバージョンの Acrobat では、そのバージョンよりも上位の互換性が設定されている PDF ドキュメントを開くことはできません。例えば、「Acrobat 7.0 以降」オプションを選択した場合は、そのドキュメントを Acrobat 6.0 およびそれより前のバージョンで開くことはできません。

互換性レベルが同じソースの PDF の互換性と一致することを確認します。例えば、標準の PDF 設定（Acrobat 5.0 以降との互換性があります）を使用するように設定された監視フォルダーがある場合は、セキュリティの互換性レベルを Acrobat 5.0 より上位にすることはできません。

**ドキュメントの制限：** 使用できるドキュメントの制限は、選択した「互換性」オプションによって異なります。

**No Encryption:ドキュメント** のどの部分も暗号化しません。

**すべてのドキュメントコンテンツを暗号化：ドキュメントとドキュメントのメタデータを** 暗号化します。このオプションを選択した場合、検索エンジンはドキュメントのメタデータにアクセスできません。

**メタデータ以外のすべてのドキュメントコンテンツを暗号化(Acrobat 6以降と互換):ドキュメントのコンテンツを** 暗号化しますが、検索エンジンはドキュメントのメタデータにアクセスできます。このオプションは、「互換性」オプションが Acrobat 6.0 以降、Acrobat 7.0 以降または Acrobat9.0 以降に設定されている場合にのみ使用できます。

**添付ファイルのみを暗号化(Acrobat 7以降互換):** ユーザーはパスワードなしでドキュメントを開くことができますが、添付ファイルを開くにはパスワードを入力する必要があります。このオプションは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されている場合にのみ使用できます。

次の設定で、パスワードのセキュリティを設定します。

>[!NOTE]
>
>パスワードを忘れた場合は、ドキュメントから復元することはできません。忘れた場合のために、パスワードは別の安全な場所に保存しておくことをお勧めします。または、パスワードで保護されていないドキュメントのバックアップコピーをとっておきます。

**ドキュメントを開く際にパスワードが必要：パスワードオプション** を有効にします。

**ドキュメントを開くパスワード：指定したドキュメントを入力しない限り、パスワードを開くことを** 禁止します。パスワードは、大文字と小文字が区別されます。Acrobat では、RSA Security Inc. の RC4 セキュリティ方式を使用して PDF ドキュメントをパスワードで保護します。印刷や編集を制限している場合は、セキュリティを強化するためにドキュメントを開くパスワードの追加をお勧めします。

**ドキュメントを開くパスワードの再入力：ドキュメントを開くパスワードが正しいこと** を確認します。

**添付ファイルを開く際にパスワードが必要：パスワードオプション** を有効にします。このオプションが使用可能になるのは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されており、「ドキュメントに関する制限」オプションが「添付ファイルのみを暗号化する」に設定されている場合のみです。

**添付ファイルを開くパスワード：添付ファイルを開く際にパスワードが必要で** あることを確認します。ユーザーはパスワードを使用しなくてもドキュメントを開くことができます。このオプションが使用可能になるのは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されており、「ドキュメントに関する制限」オプションが「添付ファイルのみを暗号化する」に設定されている場合のみです。

**Retype File Attachment：パスワードが正しいことを** 確認します。このオプションが使用可能になるのは、「互換性」オプションが Acrobat 7.0 以降または Acrobat 9.0 以降に設定されており、「ドキュメントに関する制限」オプションが「添付ファイルのみを暗号化する」に設定されている場合のみです。

次のオプションで、権限を設定します。

**ドキュメントの印刷および編集とセキュリティ設定にパスワードが必要：権限の制限を** 有効にします。

**権限パスワード：ユーザーの印刷および編集を** 制限します。ユーザーは、指定したパスワードを入力しない限りこれらのセキュリティ設定を変更することはできません。「ドキュメントを開くパスワード」で使用しているパスワードは使用できません。権限パスワードを設定すると、そのパスワードを入力したユーザーだけがセキュリティ設定を変更できます。PDF ドキュメントに両方の種類のパスワードが含まれている場合、いずれかのパスワードでドキュメントが開きますが、制限されている機能は権限パスワードでのみ設定または変更できます。PDF ドキュメントに権限パスワードのみが含まれている場合や、ユーザーがドキュメントを開くパスワードを使用してドキュメントを開いている場合は、ユーザーがセキュリティ設定を変更しようとすると、パスワードの入力を求めるプロンプトが表示されます。

**権限パスワードの再入力：権限パスワードが正し** いことを確認します。

**印刷を許可：PDFドキュメントの印刷品質を** 指定します。

**なし：ユーザーがドキュメントを印刷で** きないようにします。

**低解像度(150 dpi):** ドキュメントを150 dpi以下の解像度で印刷できます。各ページはビットマップ画像として印刷されるため、印刷速度が低下することがあります。 このオプションは、高い暗号化レベル（Acrobat 5.0、6.0、7.0 または 9.0）が選択されている場合にのみ使用できます。

**高解像度：高** 品質の印刷機能をサポートするPostScriptプリンターや他のプリンターに、高品質のベクトル出力を送ることで、任意の解像度で印刷できます。

**変更を許可：PDFドキュメントで許可する編集操作を** 定義します。

**なし：ドキュメントに変更（署名フィールドやフォームフィールドへの入力を含む）を許可し** ません。

**ページの挿入、削除、回転：ページの挿入、削除、回転、しお** りやサムネールページの作成を行うことができます。このオプションは、高い暗号化レベル（Acrobat 5.0、6.0、7.0 または 9.0）が選択されている場合にのみ使用できます。

**フォームフィールドへの入力と既存の署名フィールドへの署名：** ユーザーはフォームに入力し、電子署名を追加できます。ただし、コメントを追加したり、フォームフィールドを作成したりすることはできません。このオプションは、高い暗号化レベル（Acrobat 5.0、6.0、7.0 または 9.0）が選択されている場合にのみ使用できます。

**注釈の作成、フォームフィールドの入力および既存の署名フィールドへの署名：** ユーザーはフォームに入力し、電子署名とコメントを追加できます。

**ページレイアウト、タッチアップ、フォームフィールドへの入力、既存の署名フィールドへの署名：ユーザーは、ページの挿入、回転、削除、しおりやサムネール画像の作成、フォームの入力、電子署名の追加を** 行うことができます。このオプションでは、ユーザーはフォームフィールドを作成できません。このオプションは、低い暗号化レベル（Acrobat 3.0）が選択されている場合にのみ使用できます。

**ページの抽出を除くすべて：ページの削除を除き、Changes許可リストーの任意のドキュメントを使用してメソッドを変更で** きます。

**テキスト、画像、その他のコンテンツのコピーを有効にする：PDFドキュメントのコンテンツ** を選択してコピーできます。また、PDF ファイルのコンテンツへのアクセスを必要とするユーティリティ（Acrobat Catalog など）で、それらのコンテンツを取得することもできます。このオプションは、高い暗号化レベルが選択されている場合にのみ使用できます。

**視覚障害者向けの画面Readerデバイスのテキストアクセスを有効にする：視覚障害を持つ** ユーザーは、スクリーンリーダーを使用してドキュメントを読み取ることができます。ただし、ユーザーはドキュメントのコンテンツをコピーしたり抽出したりすることはできません。このオプションは、高い暗号化レベルが選択されている場合にのみ使用できます。

## セキュリティ設定の削除  {#delete-a-security-setting}

不要になったセキュリティ設定は削除できます。ただし、事前に設定されているセキュリティ設定は削除できません。

1. 管理コンソールで、**[!UICONTROL サービス/PDF Generator/セキュリティ設定]**&#x200B;をクリックします。
1. 削除する設定の隣にあるチェックボックスを選択します。複数の設定を選択できます。
1. 「**[!UICONTROL 削除]**」をクリックし、**[!UICONTROL 削除の確認]**&#x200B;ページで、もう一度「**[!UICONTROL 削除]**」をクリックします。
