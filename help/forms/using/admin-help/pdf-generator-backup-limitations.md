# PDF Generator バックアップの制限事項 {#pdf-generator-backup-limitations}

PDF Generatorがファイルの変換に使用する一時ディレクトリはバックアップできません。 サービスが適切に復元されても、PDF Generatorが一時ディレクトリのコンテンツを設定された間隔で確認およびクリアするので、データが失われる可能性があります。
